# ♟️ Quark ChessBot

**Quark ChessBot** is a custom-built chess engine that combines classical search algorithms with modern optimization techniques to deliver efficient and intelligent gameplay.

Developed as a focused exploration into engine design, this project emphasizes both algorithmic depth and clean implementation.

---

## 🚀 Features

* 🔍 **Alpha–Beta Pruning**
  Efficient game tree exploration using optimized minimax search.

* ⚡ **Advanced Pruning Techniques**
  Reduces unnecessary computations to improve performance.

* 🧠 **Positional Evaluation Function**
  Evaluates board states using strategic heuristics beyond material count.

* 🛠️ **Custom Engine Architecture**
  Built from scratch for full control and learning.

---

## 🎯 Project Goals

* Understand the inner workings of chess engines
* Implement efficient search algorithms
* Experiment with evaluation strategies
* Build a strong foundation for future improvements (e.g., NNUE, ML-based eval)
